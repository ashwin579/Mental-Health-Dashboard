📋 What is the Allo Health DSS?
The Decision Support System (DSS) is Allo's standardised clinical protocol — a step-by-step prescribing guide for each diagnosis. It specifies the first-line drug, bridging agent, adjuncts, assessments, and escalation triggers for each L1→L2 combination.
Allo Health DSS is evidence-based (NICE CG90/CG113, WHO mhGAP, WFSBP guidelines) and pending Dr. Sandip's formal approval before deployment to doctors. Doctor's DSS reflects actual prescribing patterns from the pilot data. Deviations from the Allo Health DSS drive doctor coaching or DSS updates — both are valid outcomes.
L1→L2 taxonomy · Each patient belongs to exactly one L1. ICD-11 codes from screening assessments added where available.

How does the backend engine work once PM marks a verdict? This tab explains the feedback loop — read before discussing with the tech team.
Verification → Confidence Update Flow
When PM marks "Confirmed MH":
1. Case added to hypothesis's confirmed_count
2. Precision = confirmed_count / total_flagged → recalculated live
3. If precision crosses 70% threshold AND supporting MH cases ≥ 5 → hypothesis auto-promotes to High
4. Confirmed case added to adjusted MH pool in category mix
5. Doctor's SH→MH detection score updated

When PM marks "Not MH":
1. false_positive_count incremented
2. Precision drops → if below 40% → hypothesis demoted to Low
3. Confirmed pool unchanged, flag removed from adjusted count

Promotion/Demotion thresholds:
Low → Medium: precision ≥40% AND ≥3 PM confirmations
Medium → High: precision ≥70% AND ≥5 PM confirmations
High → Medium: precision drops below 55% after ≥5 verdicts
Doctor-Level Variance Engine
High-Variance Detection (triggers coaching alert):
If Doctor A's SH→MH rate for a complaint type is >2× Doctor B's rate on the same complaint type → flagged as high variance

What gets discussed:
• Is it patient-mix difference? (age, severity skew)
• Is it assessment depth difference? (screening form filled vs skipped)
• Is it clinical skill difference? (probe questions asked)

DSS alignment check:
Doctors whose detection rate aligns with high-confidence hypotheses get a "Pattern-consistent" badge. Outliers get a coaching flag.

When to update the DSS:
If 3+ doctors independently detect the same SH→MH pattern at >70% precision → pattern is promoted to DSS SH screening question
